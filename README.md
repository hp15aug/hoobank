# HooBank
A Clone of a website for practicing React and Tailwind CSS(Frontend).

![HooBank](https://i.ibb.co/BK1Hn0x/Screenshot-2022-08-08-at-4-05-48-PM.png)


## Deployment

To deploy this project run

```bash
  npm run dev
```

will run at server
```bash
  http://localhost:5173/
```

## Tech Stack

**Client:** React, Tailwind Css

**Server:** Node, Express
